## Como executar localmente

Opções simples:

1. Abrir o arquivo `index.html` diretamente no navegador.

2. Servir em um servidor local (recomendado para evitar problemas com requisições externas):

   - Com Python 3:
     ```bash
     python3 -m http.server 8000
     ```
     Acesse: http://localhost:8000

   - Com Node (serve):
     ```bash
     npx serve .
     ```
